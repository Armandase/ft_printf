
# Printf

42's printf project

Re-code the libc's printf function in c.

## Usage/Examples

Basic usage:

```c
ft_printf("string to print on the terminal");
```

Usage with parameters:
```c
ft_printf("string: %s int: %d", str, value);
```
where str is a char * and value and int


## Features

- %s -> char *
- %d -> int
- %x -> int to hex
- %u -> unsigned int
- %c -> char
- %p -> pointer
- %% -> percent sign
